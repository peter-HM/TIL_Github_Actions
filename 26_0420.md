#### TIA (Today I asked)

- port 80? :

브라우저는 URL에 포트가 없으면 프로토콜을 기본 프토로 자동 연결을 함.

ex) - http://example.com -> http://example.com:80
    - https://example.com -> https://example.com:443


- gitignore vs dockerignore

.gitignore: Github에 올리지 않을 파일. git push가 기준

.dockerignore: Docker image에 포함하지 않을 파일. docker build 할때. Docker Hub에 DB PW 등을 안전하게 지키기위해.

- .env

.env 없이 뜰 수 있는 지 확인하는 방법. 추적하면 된다. .env.example을 가지고


```

# 개발할 때
git checkout dev
# 코드 수정 후
git add . && git commit -m "..."
git push origin dev

# 배포할 때
git checkout main
git merge dev
git push origin main
# → GitHub Actions 자동 실행!

```
